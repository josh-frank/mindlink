## MindLink

### EDA & polygraph stack for RaspberryPi

With a Grove HAT, turn your RaspberryPi into a EDA measurement/recording instrument

#### Features (*upcoming features starred in italics**)
- EDA measurement pipeline: broadcasting with flask/websockets
- EDA metering instrument interface: GUI in HTML/JS
- *CSV/SRT save function**
- *timestamping feature**
- *audio recording**
- *pulse oximeter, rPPG and other sensors**
- *machine learning API**
